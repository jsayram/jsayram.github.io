What new technologies are you currently researching?

Im currently working on learning and understanding Hibernate & Spring frameworks. Hibernate is a persistence framework and is used to persist Java objects in a relational database. Spring can be used to wire all this together and to provide declarative transaction management. I believe understanding and working with this frameworks is essential for becoming a well rounded Software Engineer
